# Cryptocurrency Price Tracker

## Project Description

Cryptocurrency Price Tracker is a Python-based application that fetches real-time cryptocurrency information using the CoinGecko API.

The application allows users to search for cryptocurrencies, view detailed market information, display the top 10 cryptocurrencies by market capitalization, and save search results to a CSV file.

---

## Features

* Search Cryptocurrency
* Display Current Price
* Display Symbol
* Display Market Cap Rank
* Display 24-Hour Price Change
* View Top 10 Cryptocurrencies
* Error Handling
* Invalid Coin Detection
* Export Data to CSV

---

## Technologies Used

* Python
* Requests Library
* CoinGecko API
* CSV Module
